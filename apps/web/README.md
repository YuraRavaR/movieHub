# MovieHub Web

Frontend app for MovieHub, built with Next.js.

## Run (from repo root)

```bash
npm run dev
```

Web app is available at `http://localhost:3001`.

## Notes

- Uses API base URL from `apps/web/.env.local` (`NEXT_PUBLIC_API_URL`).
- Main pages:
  - `/` home
  - `/search` movie search
  - `/watchlist` user movie list

For full setup instructions, see root `README.md`.
